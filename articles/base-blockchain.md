# Base

**Base** is an [[Ethereum]] [[Layer 2]] rollup built on the [[OP Stack]], incubated by [[Coinbase]]. Launched in August 2023, Base has emerged as a leading platform for [[AI agents]], consumer applications, and the broader "onchain economy." The network combines Coinbase's distribution with Ethereum's security and the Optimism ecosystem's shared infrastructure.

## Overview

Base represents Coinbase's strategic commitment to bringing the next billion users onchain. Unlike many crypto projects, Base has no native tokenâ€”users pay transaction fees in ETH. The network operates as an [[optimistic rollup]], posting transaction data to Ethereum mainnet while offering significantly lower fees and higher throughput.

Key characteristics:
- **Low fees**: Typically under $0.01 per transaction
- **High throughput**: Target of 1+ gigagas per second
- **EVM equivalent**: Full compatibility with Ethereum tooling
- **Coinbase integration**: Direct onramps from Coinbase exchange

## Technical Architecture

### OP Stack Foundation

Base is built on the [[OP Stack]], the modular open-source framework that also powers [[Optimism]] mainnet. This provides:

- Shared sequencer potential through the Superchain vision
- Battle-tested codebase with billions secured
- Continuous improvements from Optimism Collective R&D
- Interoperability with other OP Stack chains

### Network Specifications

**Parameter / Value:**

- **Block time**: 2 seconds
- **Chain ID**: 8453
- **Settlement**: Ethereum mainnet
- **Data availability**: Ethereum (calldata/blobs)

### EIP-4844 Integration

Base was among the first chains to adopt [[EIP-4844]] (Proto-Danksharding), dramatically reducing data availability costs through blob transactions. This enabled sub-cent transaction fees for most operations.
